# Água Jump Studios — Homelab System

AJShomelab is a lightweight tool that turns your PC into a personal server with basic authentication support.

---

## Overview

This project is intended for experimentation and local/self-hosted setups.
Use it to quickly spin up a personal server environment without complex configuration.

---

## Requirements

* Internet connection (4g to 5g)
* Windows system with `.exe` support for this beta
* Minimum **5 GB RAM** *(subject to change)*
